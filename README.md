🔐 Password Strength Checker (Python)

A simple and efficient Password Strength Checker built using Python. This project evaluates the strength of a user-entered password based on essential security criteria and provides feedback to improve weak passwords.

🚀 Features

Checks minimum password length (8+ characters)

Validates presence of:

Lowercase letters

Uppercase letters

Numbers

Special characters

Assigns a strength score

Categorizes password as:

Very Weak

Weak

Medium

Strong

Very Strong

Provides improvement suggestions for missing criteria

🛠 Technologies Used

Python

re (Regular Expressions module)

📌 How It Works

The program uses regular expressions to analyze the password against five security criteria. Each satisfied condition increases the score. Based on the total score, the password strength is classified and improvement tips are displayed if needed.

🎯 Purpose

This project demonstrates:

Input handling

Conditional logic

Regular expressions

Basic security validation principles

It is ideal for beginners learning Python and understanding how password validation systems work.
